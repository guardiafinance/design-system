Select

import Select from "ui_kits/components/Select"

Dropdown nativo. Ideal para listas curtas e conhecidas (até ~12 opções). Para listas longas, busca ou criação de novos valores, use Combobox.

PropTipoDefaultDescrição

Usar
  • Lista curta (< 15 itens) sem necessidade de busca.
  • Placeholder indicando o que selecionar.
Evitar
  • Listas longas — use Combobox.
  • Selects dependentes (cascata) sem feedback de carregamento.
