Accordion

import Accordion from "ui_kits/components/Accordion"

Grupos expansíveis para conteúdo denso: plano de contas, filtros avançados, detalhes de lançamento, configurações. type="single" permite só um aberto por vez; multiple permite vários.

PropTipoDefaultDescrição
itemsarray—{`{ id, title, content, icon?, defaultOpen?, disabled? }[]`}
typestring"multiple"single · multiple
variantstring"bordered"bordered · plain
Usar
  • Agrupe conteúdo longo e secundário que o usuário consulta sob demanda — FAQs, filtros avançados, detalhes de conciliação.
  • Mantenha o primeiro item aberto quando for o caminho mais comum.
Evitar
  • Esconder conteúdo crítico (erros, alertas, ações primárias).
  • Usar em fluxos lineares onde o usuário precisa ver tudo na ordem.
