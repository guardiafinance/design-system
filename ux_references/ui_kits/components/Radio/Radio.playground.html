Usar
  • Escolha única entre 2-5 opções visíveis simultaneamente.
  • Label clicável; opção default pré-selecionada quando fizer sentido.
Evitar
  • Mais de 5 opções — use Select.
  • Radios soltos sem fieldset/legend.
