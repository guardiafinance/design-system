Alert

import Alert from "ui_kits/components/Alert"

Banner inline para comunicar status, aviso ou erro. Quatro tipos: info, success, warning, danger. Admite título, ação e botão fechar.

PropTipoDefaultDescrição
typestring"info"info · success · warning · danger
titlenode—Título em negrito
childrennode—Mensagem
iconstring | nullautoNome Lucide, ou null para ocultar
actionnode—Botão/link à direita
closablebooleanfalseMostra X para dispensar
onClose() ⇒ void—Callback do botão fechar
Usar
  • Mensagem persistente em banner quando a ação do usuário é necessária — divergência fiscal, consentimento, erro bloqueante.
  • Sempre pares ícone + título + descrição curta; action opcional à direita.
Evitar
  • Confirmações transitórias ("salvo com sucesso") — use Toast.
  • Vários alerts empilhados competindo pela atenção.
