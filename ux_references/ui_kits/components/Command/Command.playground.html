Paleta de comandos com busca fuzzy, grupos, ícones, atalhos e navegação por teclado (↑/↓/↵/Esc).

PropTipoDefaultDescrição
open · onCloseboolean · fn—Controle
itemsCmdGroup[]—Grupos com entradas
placeholderstring"Buscar comando…"Texto da busca
emptyTextstring"Nenhum resultado"Estado vazio
CmdEntry
id · labelstring · node—Identificador + rótulo
icon · shortcutstring—Ícone Lucide · atalho exibido
keywordsstring—Termos extra para a busca
actionfn—Callback ao executar
Usar
  • Palette de comando (⌘K) para power users: navegar rápido, disparar agente, abrir cliente.
  • Agrupar por seção ("Navegação", "Agentes", "Ações").
Evitar
  • Como substituto da navegação principal.
  • Comandos sem atalho de teclado visível.
