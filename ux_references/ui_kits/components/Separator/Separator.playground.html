Separator

import Separator from "ui_kits/components/Separator"

Divisor horizontal ou vertical para agrupar visualmente blocos de conteúdo. Três aparências e label opcional centralizado.

PropTipoDefaultDescrição
orientationstring"horizontal"horizontal · vertical
appearancestring"solid"solid · dashed · dotted
labelstring—Texto centralizado entre dois traços
Usar
  • Dividir grupos de conteúdo relacionados dentro de cards/drawers.
  • Horizontal 1px cinza-100; vertical no ButtonGroup.
Evitar
  • Separadores como ornamentação.
  • Entre cada item de uma lista — use padding.
