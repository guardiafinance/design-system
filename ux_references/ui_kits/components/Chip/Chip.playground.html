Chip

import Chip from "ui_kits/components/Chip"

Token selecionável ou removível para filtros ativos, categorias e seleção múltipla. Diferente de Badge (passivo), o Chip é interativo.

PropTipoDefaultDescrição
selectedbooleanfalseEstado selecionado (destaque violeta)
leftIconIconName—Glifo à esquerda do label
onRemove() => void—Quando presente, mostra botão × no final
disabledbooleanfalseBloqueia interação
onClickfn—Para toggles, combine com selected
Usar
  • Filtros removíveis ativos (ex.: "Itaú ×", "Pendentes ×") e tags editáveis.
  • Ícone x à direita quando removível.
Evitar
  • Uso puramente decorativo — isso é Badge.
  • Chips estáticos sem interação.
