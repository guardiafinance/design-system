Select com busca — para listas longas (clientes, contas, CNAEs). Aceita metadado por item e estado limpável.

Usar
  • Seleção em lista grande com busca — plano de contas, fornecedores, CFOPs.
  • Mostrar chave + descrição ("3.1.4.05 · Despesas com tecnologia").
Evitar
  • Listas curtas (< 7 itens) — use Select simples.
  • Combobox sem placeholder indicando o que se busca.
