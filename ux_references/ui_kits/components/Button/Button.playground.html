Button

import Button from "ui_kits/components/Button"

Ação interativa principal. Seis variantes, três tamanhos, slots para ícones e estado loading. Primário é o violeta; accent laranja é reservado para conversão; outline e ghost para ações secundárias.

PropTipoDefaultDescrição
variantstring"primary"primary · accent · outline · ghost · destructive · link
sizestring"md"sm · md · lg
leftIconIconName—Nome do glifo (de <Icon />)
rightIconIconName—Idem, à direita do label
loadingbooleanfalseMostra spinner e desabilita clique
fullWidthbooleanfalseOcupa 100% do container
disabledbooleanfalseDesabilita interação
Usar
  • Uma ação primária por tela — reserve accent laranja para conversão/CTA hero.
  • Label em sentence case com verbo de ação ("Salvar lançamento", não "Salvar").
  • Destructive apenas para ações irreversíveis com confirmação.
Evitar
  • Dois botões primários competindo na mesma tela.
  • Labels vagos ("OK", "Enviar") sem objeto.
  • Gradientes ou sombras no botão — a hierarquia é por cor sólida e peso.
