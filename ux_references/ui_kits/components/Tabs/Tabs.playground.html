Tabs

import Tabs from "ui_kits/components/Tabs"

Alternância entre views dentro de uma mesma tela. Três variantes: underline (padrão, para tabs de página), pills (filtros leves) e boxed (seções compactas).

PropTipoDefaultDescrição
itemsarray—{`{ value, label, icon?, badge?, disabled? }[]`}
value · defaultValuestring—Tab ativa controlada/inicial
onChange(v) ⇒ void—Callback da troca
variantstring"underline"underline · pills · boxed
sizestring"md"sm · md
Usar
  • Visões paralelas do mesmo recurso (Resumo / Lançamentos / Auditoria de um cliente).
  • 2-5 tabs; primeira como padrão.
Evitar
  • Navegação principal — use SidebarNav.
  • Tabs com badge de contagem fora da paleta.
