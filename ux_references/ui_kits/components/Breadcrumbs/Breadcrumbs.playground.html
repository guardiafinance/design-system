Breadcrumbs

import Breadcrumbs from "ui_kits/components/Breadcrumbs"

Trilha de migalhas para orientação em hierarquias profundas. Último item é o atual (não é link). Use maxItems para truncar caminhos longos.

PropTipoDefaultDescrição
itemsarray—{`{ label, href?, icon?, onClick? }[]`}
separatornodechevron-rightSeparador entre itens
maxItemsnumber—Trunca com "…" quando excedido
Usar
  • Navegação hierárquica em fluxos profundos: Cliente › Empresa › Conciliação › Lote.
  • Último item como estado atual (não clicável).
Evitar
  • Breadcrumbs para histórico linear (use Tabs ou volta).
  • Mais de 5 níveis — trunque com ellipsis.
