ConfidenceIndicator

import ConfidenceIndicator from "ui_kits/components/ConfidenceIndicator"

Comunica a confiança da Guardia em um output de IA. Três níveis alinhados ao sistema Lighthouse — alta (≥95%), revisar (80–94%), atenção (<80%). Variantes: chip, bar, dot.

PropTipoDefaultDescrição
valuenumber—0–100; deriva o nível se level não for passado
levelstring—high · medium · low (sobrepõe value)
variantstring"chip"chip · bar · dot
showValuebooleantrueExibe % (quando value é passado)
labelnodeautoRótulo custom; default: "Alta confiança" / "Revisar" / "Atenção"
Usar
  • Sempre com contexto textual (Alta / Revisar / Atenção), nunca só cor.
  • Verde ≥ 95% · laranja 80-94% · vermelho < 80% (thresholds configuráveis por cliente).
Evitar
  • Cor isolada sem número nem label.
  • Thresholds inconsistentes entre telas do mesmo produto.
