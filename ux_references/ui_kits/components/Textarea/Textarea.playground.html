FORM · ENTRADA MULTI-LINHA

Textarea

import Textarea from "ui_kits/components/Textarea"

Campo multi-linha. Suporta size (sm/md/lg), estados (default/error/success), controle de resize e contador de caracteres via showCount + maxLength.

PropTipoDefaultDescrição
sizestring"md"sm · md · lg
statestring"default"default · error · success
invalidbooleanfalseForça estado error
resizestring"vertical"none · vertical · both
showCountbooleanfalseExibe contador (use com maxLength)
maxLengthnumber—Limite nativo
Usar
  • Texto multilinha: observações em lançamento, notas de conciliação.
  • Indicador de caracteres se houver limite.
Evitar
  • Textarea para campos curtos — use Input.
  • Sem indicação de altura mínima.
