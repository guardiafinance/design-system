Skeleton

import Skeleton from "ui_kits/components/Skeleton"

Placeholder animado (shimmer) para conteúdo que ainda está carregando. Variantes: text, title, rect, circle. Para blocos de texto multi-linha, passe lines.

PropTipoDefaultDescrição
variantstring"text"text · title · rect · circle
widthnumber | stringautoLargura (px ou %)
heightnumber | stringautoAltura (px ou %)
linesnumber1Número de linhas (só com variant=text); última linha a 70%
Usar
  • Loading de listas e cards quando a forma do conteúdo é previsível.
  • Mesmo shape do conteúdo final; animação sutil.
Evitar
  • Skeleton para requests instantâneos (< 300ms).
  • Loader genérico no meio da tela quando dá pra mostrar estrutura.
