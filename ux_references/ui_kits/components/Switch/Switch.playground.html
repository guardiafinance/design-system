Switch

import Switch from "ui_kits/components/Switch"

Alternador on/off para ações com efeito imediato — prefira sobre checkbox quando a mudança afeta estado do sistema (notificações, integrações, automações). Dois tamanhos, label e descrição opcionais.

PropTipoDefaultDescrição
sizestring"md"sm · md
labelReactNode—Texto principal ao lado do toggle
descriptionReactNode—Texto de apoio abaixo do label
checkedboolean—Uso controlado
defaultCheckedboolean—Uso não-controlado
disabledbooleanfalseBloqueia interação
Usar
  • Liga/desliga aplicado imediatamente (auto-conciliação, notificações).
  • Label à esquerda; estado on = laranja.
Evitar
  • Configurações que precisam "salvar" — use Checkbox.
  • Switch para ação temporária (filtros).
