Campo de texto de linha única. Suporta ícones à esquerda/direita, affix textual (R$, @, %), três tamanhos e três estados semânticos (default, erro, sucesso).

PropTipoDefaultDescrição

suffixReactNode—Texto afixado no fim (%, .com.br)

Usar
  • Campos de texto livre, busca, numéricos financeiros com máscara.
  • Label sempre visível; hint abaixo para formato; erro inline vermelho.
Evitar
  • Placeholder como substituto de label.
  • Mensagens de erro genéricas ("Inválido").
