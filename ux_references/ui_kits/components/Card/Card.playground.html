CONTAINER · CARD

Card

import Card from "ui_kits/components/Card"

Contêiner base com padding, borda e sombra. Composição via subcomponentes: Card.Header, Card.Body, Card.Footer.

PropTipoDefaultDescrição
variantstring"outlined"plain · outlined · elevated · filled
paddingstring"md"none · sm · md · lg
interactivebooleanfalseAdiciona hover + cursor pointer
onClick() ⇒ void—Torna o Card um botão
Usar
  • Container de uma unidade de informação — lançamento, cliente, relatório.
  • Borda sutil + padding 18-24px + sombra mínima.
Evitar
  • Cards aninhados dentro de cards.
  • Sombras pesadas ou bordas coloridas como acento padrão.
