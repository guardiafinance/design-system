PRIMITIVO · FORMULÁRIO

Label

import Label from "ui_kits/components/Label"

Rótulo semântico para campos de formulário. Suporta indicação de obrigatório (*) ou opcional. Sempre usar htmlFor apontando para o id do input.

PropTipoDefaultDescrição
htmlForstring—Id do input associado (acessibilidade)
requiredbooleanfalseMostra asterisco vermelho
optionalbooleanfalseMostra sufixo "(opcional)"
Usar
  • Antecede cada input/select/textarea; clicável; indicador * para obrigatório.
  • Texto conciso, sentence case.
Evitar
  • Labels em caixa alta ou longas demais.
  • Esconder label em nome de minimalismo.
