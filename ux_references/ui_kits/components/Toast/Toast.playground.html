Toast

import { ToastProvider, useToast } from "ui_kits/components/Toast"

Notificações transitórias, empilhadas no canto. Uso: envolva o app em <ToastProvider> e chame toast.show() via hook.

toast.show(options)
OpçãoTipoDefaultDescrição
typestring"info"info · success · warning · danger
titlenode—Linha principal
descriptionnode—Apoio
durationnumber4200ms; 0 = persistente
action{`{ label, onClick }`}—Botão à direita; fecha ao clicar
ToastProvider
positionstring"bottom-right"top-right · bottom-right · top-left · bottom-left
Usar
  • Confirmação transitória não-bloqueante ("lançamento salvo", "XML importado").
  • Auto-dismiss em 4-6s; ação "Desfazer" quando aplicável.
Evitar
  • Erros que exigem atenção — use Alert.
  • Empilhar > 3 toasts simultâneos.
