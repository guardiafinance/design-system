# Architecture: Plan #208 — Invert `--primary` / `--secondary` in Light Mode

## Design Decision

**Token-layer inversion at a single locus.** Every visible component consuming `--primary` / `--secondary` (and aliases `--accent`, `--ring`, `--*-foreground`) inherits the new mapping with zero per-component change. This is the minimal-surface, maximum-cascade path. No new ADR is opened: the only architectural justification for the previous mapping was `shadcn`-compat continuity, which Brand canonicity (`lex-brand-colors` / Notion) supersedes. Rationale is recorded inline in the CSS itself.

## Contrast Analysis (lex-frontend-accessibility, lex-brand-colors)

| Combination | Ratio | WCAG | Use |
|---|---|---|---|
| White (`#FDFDFD`) on Violet-500 (`#4F186D`) — `--primary` background | **7.85:1** | AAA Normal | Button.default, IconButton.default, Switch checked, Input border (UI ≥3:1), Calendar day-selected, Typography variant primary text. **No regression** — AAA. |
| White (`#FDFDFD`) on Orange-500 (`#E07400`) — `--secondary` background | **3.36:1** | AA Large only (≥18px or 14px bold) | Button.secondary, IconButton.secondary, Sheet close-state. These are **button/UI surfaces** explicitly admitted by `lex-brand-colors` "Combinations in the 3:1–4.5:1 range are restricted to titles, buttons, and badges." **Status quo preserved**: previously `bg-primary` (orange) carried this caveat — the relaxed-band exception simply moves from `bg-primary` to `bg-secondary`. No new violation. axe color-contrast disables at meta level in Button.stories.tsx already cover this case. |
| Violet-500 (`#4F186D`) text underline on White — `text-primary` (link variant) | **7.85:1** | AAA Normal | Button.link, Typography variant primary. Improvement vs. previous orange-500-on-white (3.36:1). |
| `--ring` (Violet-500) focus ring against white background | **7.85:1** | AAA (UI ≥3:1) | Focus rings throughout. Improvement vs. previous orange. |

**Conclusion:** the inversion does not introduce any new violation; it improves the dominant CTA contrast from AA-Large (3.36:1) to AAA Normal (7.85:1) and moves the lower-band combination from `--primary` to `--secondary` (a lower-frequency surface).
